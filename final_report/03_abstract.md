# Abstract

The vehicle spare parts market in Sri Lanka remains fragmented and inefficient, with buyers and sellers relying on general-purpose online marketplaces and social media platforms that lack specialised tools for part compatibility verification, image-based identification, and price guidance. This project addresses these challenges by developing SPAREHUBLK, an AI-powered web-based marketplace dedicated to automobile spare parts trading in Sri Lanka.

The system was developed as a full-stack web application using the MERN stack, comprising MongoDB for data persistence, Express.js and Node.js for the backend API, and React with Vite for the frontend user interface. The platform provides user registration and authentication, structured spare part listings with detailed vehicle compatibility information, advanced search and filtering, seller profiles with a premium tier system, product and seller reviews, and an admin dashboard for platform management.

The intelligent layer of the system is powered by Google Gemini AI, which provides four key capabilities: a conversational chatbot for spare parts queries, image-based part identification, VIN and chassis number decoding for vehicle compatibility, and market intelligence for price analysis. These features were integrated to reduce the cognitive load on users who may lack technical expertise in part identification or market pricing.

The system was evaluated through functional testing, usability assessment, and performance benchmarking. Results indicate that SPAREHUBLK successfully streamlines the spare parts discovery process, improves listing quality through structured data entry, and provides meaningful AI-assisted guidance to users. While the project achieved its primary objectives, some advanced features remain open for further refinement, particularly in expanding the AI training data for more diverse part categories.

In conclusion, SPAREHUBLK demonstrates how integrating modern web technologies with accessible AI services can address real-world marketplace inefficiencies. The platform offers a more focused, user-friendly solution for spare parts trading compared to general-purpose alternatives, contributing both practically to the local automotive community and academically to the field of AI-enhanced e-commerce systems.

**Keywords:** Artificial Intelligence, E-Commerce, MERN Stack, Spare Parts Marketplace, Image Recognition, Price Analysis, Sri Lanka
